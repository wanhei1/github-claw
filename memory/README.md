# memory/

This directory stores **temporary** process notes, task records, and drafts for the AI assistant.

## Purpose

- In-progress task notes and intermediate results.
- Drafts that are not yet ready to be promoted to `MEMORY.md`.
- Scratch space for exploration and reasoning during a task.

## Structure

| Path | Purpose |
|------|---------|
| `memory/daily/` | Daily logs, one file per day (`YYYY-MM-DD.md`). |

## Lifecycle

- Temporary notes in this directory should be cleaned up or archived after the related task is complete.
- Conclusions with long-term value should be **moved or summarized into `MEMORY.md`**.
- Do not store sensitive information (passwords, tokens, keys) here.
