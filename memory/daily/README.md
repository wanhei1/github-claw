# memory/daily/

This directory stores **daily logs** for the AI assistant and the repository owner.

## Naming Convention

Each daily log file must be named using the `YYYY-MM-DD.md` format, for example:

```
2026-05-03.md
2026-05-04.md
```

## What to Record

A daily log may contain:

- Tasks started or completed that day.
- Key decisions made.
- Open questions or blockers.
- Links to relevant commits, PRs, or issues.
- Short notes that don't yet belong in `MEMORY.md`.

## Lifecycle

- Daily logs are temporary. After the relevant work is finished, summarize lasting conclusions into `MEMORY.md`.
- Old daily logs may be archived or deleted once their content has been promoted or is no longer relevant.
- Do not store sensitive information (passwords, tokens, keys) in daily logs.
